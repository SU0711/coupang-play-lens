# [Coupang Play Lens](https://coupang-play-lens.vercel.app)

쿠팡 플레이 시청 중 상품을 길게 누르면 팝업을 띄워 쿠팡 내 관련 쇼핑 정보 및 간편 결제를 제공하는 서비스입니다.

소비자의 구매 여정 간소화, 자사 플랫폼간 연동 및 구매율 증진을 목표로 합니다.
